# **3択暗記システム**

## **概要**

3択クイズができるプログラムです。
自分でクイズを登録できます。

3つのモードがあります。

1つ目は、"問題を登録する"モードです。自分が解きたい問題を登録することができます。

2つ目は、"覚える"モードです。登録されている問題を一覧で表示できます。答えはオレンジで表示されるため、赤シートで隠すことができます。

3つ目は、"テスト"モードです。
登録した問題が3択クイズで遊べます。
出題する問題数、出題の順番などを設定できます。
また、問題ごとに正解/不正解、最後には正解した問題数や正答率が表示されます。

## **操作方法**
1つ目の"問題を登録する"モードでは、5つの項目を入力してください。

入力する項目は、問題文/選択肢①/選択肢②/選択肢③/答え の5つです。

"問題を登録する"ボタンを押してください。入力ボックスの上に書かれている指示を見ながら、それぞれの項目について入力し、送信ボタンを押してください。

答え以外の4項目は文章、単語、数値などどんな形でも構いません。

その後、自らが入力した選択肢が番号とともに表示されます。そこから答えと対応する半角数字を入力してください。半角数字以外を入力した場合、再入力となります。答えとなる文字列では入力できません。
